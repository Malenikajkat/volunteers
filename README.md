# README

## Описание проекта
Проект представляет собой систему для управления волонтёрским движением. Программное решение автоматизирует процессы регистрации участников, ведения базы волонтёров, распределения задач, учёта отработанного времени и формирования отчётов.

## Цель разработки
Создать удобный инструмент для организации и контроля волонтёрской деятельности, который позволит:
- вести учёт волонтёров и их специализаций;
- назначать задачи и отслеживать их выполнение;
- автоматически учитывать отработанные часы;
- формировать и экспортировать отчёты по активности волонтёров и задачам.

## Технические требования
- **Язык программирования:** Python.
- **Интерфейс:** PyQt 6.
- **Срок реализации:** 4 часа.
- **Сохранение данных:** автоматическое сохранение всех изменений.

## Функциональные возможности

### 1. Регистрация и вход
- Регистрация новых пользователей с вводом ФИО, электронной почты, телефона и пароля.
- Вход в систему по e-mail и паролю.

### 2. База волонтёров
- Добавление, редактирование и удаление записей о волонтёрах (имя, возраст, контакты, специализация).
- Поиск по базе по ФИО и специализации.

### 3. Распределение задач
- Создание задач с описанием, сроками и требуемым числом волонтёров.
- Назначение волонтёров на задачи.
- Просмотр назначенных задач для каждого волонтёра.

### 4. Учёт отработанных часов
- Фиксация времени работы каждого волонтёра по каждой задаче.
- Автоматический подсчёт общего количества часов для каждого участника.

### 5. Формирование отчётов
- Отчёты по активности волонтёров (число выполненных задач, общее время).
- Сводные отчёты по задачам (название, сроки, исполнители, потраченное время).

### 6. Экспорт отчётов
- Экспорт отчётов в PDF и Excel.

### 7. Защита от ошибок ввода
- Минимизация некорректного ввода данных пользователем.

### 8. Стабильность и интерфейс
- Программа запускается быстро и работает стабильно.
- Интерфейс интуитивно понятен и эстетичен.
