Техническое задание
Название проекта:
Разработка системы управления волонтерским движением
Цель разработки:
Создание программного продукта для организации и учета волонтерской деятельности, включающего регистрацию пользователей, базу волонтеров, распределение задач, учет отработанных часов и формирование отчетов.
Основные требования:
Общие требования:
•	Система должна разрабатываться на Python с использованием библиотеки PyQt 6.
•	Проект выполняется за 4 часа.
•	Интерфейс программы должен быть интуитивно понятным и удобным для пользователя.
•	Все изменения сохраняются автоматически.
Функциональные требования:
1.	Регистрация и вход пользователя
o	Пользователь должен иметь возможность зарегистрироваться в системе, указывая ФИО, электронную почту, телефон и пароль.
o	Реализовать механизм входа в систему по электронной почте и паролю.
2.	База волонтеров
o	Возможность добавления новых волонтеров с указанием имени, возраста, контактных данных и специализации.
o	Поиск волонтеров по различным параметрам (ФИО, специальность).
o	Редактирование и удаление записей о волонтерах.
3.	Распределение задач
o	Добавление новых задач с описанием, сроками исполнения и требуемым количеством волонтеров.
o	Назначение конкретных волонтеров на выполнение задач.
o	Отображение списка назначенных задач каждому волонтеру.
4.	Учет часов работы
o	Учет количества часов, проведенных каждым волонтером на выполнении каждой задачи.
o	Автоматическое начисление общего количества часов каждому волонтеру.
5.	Формирование отчетов
o	Генерация отчета по активности каждого волонтера (количество выполненных задач, общее число отработанных часов).
o	Формирование сводного отчета по выполненным задачам (название задачи, сроки, исполнители, потраченное время).
Дополнительные требования:
•	Минимизация ошибок ввода данных пользователями.
•	Возможность экспорта отчетов в форматы PDF и Excel.
